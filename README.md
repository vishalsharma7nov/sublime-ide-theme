# Mariana Dual Theme (Classic + Pro)

A carefully crafted VS Code / Cursor theme inspired by the original **Mariana** color scheme from Sublime Text.

This extension provides two variants:

### 🌑 Mariana Classic

A faithful recreation of the original Mariana theme, preserving its soft contrast and balanced color palette for a familiar Sublime Text experience.

### ⚡ Mariana Pro

An enhanced version optimized for modern development, especially Flutter and Dart. It improves syntax distinction, readability, and long-session comfort while staying true to Mariana’s aesthetic.

---

## ✨ Features

* 🎨 Accurate Mariana-inspired color palette
* 🧠 Improved syntax highlighting (functions, classes, variables clearly separated)
* 🚀 Optimized for Flutter & Dart development
* 🌙 Comfortable contrast for long coding sessions
* ⚡ Works seamlessly with both VS Code and Cursor

---

## 🖥 Installation

### From VSIX

1. Open Command Palette (`Cmd + Shift + P`)
2. Run: `Extensions: Install from VSIX`
3. Select the downloaded `.vsix` file

---

## 🎯 Usage

1. Open Command Palette (`Cmd + Shift + P`)
2. Select `Preferences: Color Theme`
3. Choose:

   * **Mariana Classic**
   * **Mariana Pro**

---

## 🔧 Recommended Settings

For best experience, pair the theme with:

```json
{
  "editor.fontFamily": "JetBrains Mono, Menlo, monospace",
  "editor.fontLigatures": true,
  "editor.lineHeight": 1.6,
  "editor.letterSpacing": 0.3
}
```

---

## 📌 Notes

This theme is inspired by Sublime Text’s Mariana but adapted for VS Code’s token system, ensuring better compatibility and modern language support.
